# Shell Plugin Icons

This plugin requires two icon files:

## Windows Icon: `cmd_icon.png`
- **Size:** 128x128 pixels recommended
- **Description:** Should look like the Windows Command Prompt icon
- **Location:** Place in this directory (`src/deda/plugins/shell/`)
- **Source:** You can extract the cmd.exe icon from Windows or create a custom icon that resembles the Windows command prompt

## Linux/macOS Icon: `shell_icon.png`
- **Size:** 128x128 pixels recommended
- **Description:** Should look like a terminal/shell icon (typically a terminal window or shell prompt symbol)
- **Location:** Place in this directory (`src/deda/plugins/shell/`)
- **Source:** Common terminal icons are available from icon libraries, or you can use a generic terminal/shell icon

## Temporary Solution
Until icons are added, the plugin will fall back to the default plugin icon from the plugin manager.
